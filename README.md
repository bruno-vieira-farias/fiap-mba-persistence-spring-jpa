# Java Persistence

 Trabalho final para a disciplina de Persistência em Java (JPA, Spring Data e NoSQL) <p> 
 Mininstrada por Prof. MSc. Rafael Matsuyama <p>

### Requisitos

- JDK
- Java 
- Spring
- Swagger

### Run
Execute na raiz do projeto a task `bootRun` do gradle.

- Windows
  `mvnw spring-boot:run`
- Linux / Mac
  `./mvnw spring-boot:run`
